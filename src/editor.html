This is will make all images linked anywhere in the project be stored directly in the .dmdb file.

Please be aware, this will make the project take longer to save and be larger, but there will be no broken links.

This will clamp all images resolution based off the width, the height will scale accordingly, input into the field the width you would like to use, and enable compression.
